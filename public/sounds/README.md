# BGM（背景音楽）の設定方法

このフォルダにオリジナルのBGMファイルを配置してください。

## 必要なファイル

以下の2つのファイルを配置してください：

1. **title-bgm.mp3** - タイトル画面で再生される音楽
2. **game-bgm.mp3** - ゲームプレイ中に再生される音楽

## 推奨形式

- **形式**: MP3（.mp3）またはOGG（.ogg）
- **ビットレート**: 128kbps - 192kbps
- **音量**: 適切にノーマライズされた音量
- **ループ**: BGMは自動的にループ再生されます

## 音量調整

音楽の音量はデフォルトで30%（0.3）に設定されています。
音量を変更したい場合は、`src/App.jsx`の以下の部分を編集してください：

```javascript
const titleBGM = useBackgroundMusic('/sounds/title-bgm.mp3', 0.3); // 0.3 = 30%
const gameBGM = useBackgroundMusic('/sounds/game-bgm.mp3', 0.3);   // 0.3 = 30%
```

## 使用例

```
trypas-web/
└── public/
    └── sounds/
        ├── title-bgm.mp3    ← タイトル画面用
        ├── game-bgm.mp3     ← ゲーム用
        └── README.md        ← このファイル
```

## 機能

- **自動再生**: タイトル画面やゲーム画面に入ると自動的に適切なBGMが再生されます
- **ミュート機能**: 画面右上のボタンでいつでもミュート/アンミュートできます
- **シームレスな切り替え**: 画面遷移時に自動的にBGMが切り替わります
- **ループ再生**: BGMは自動的にループします

## トラブルシューティング

### 音楽が再生されない場合

1. ファイル名が正確か確認してください（title-bgm.mp3、game-bgm.mp3）
2. ファイルがこのフォルダに配置されているか確認してください
3. ブラウザのコンソールを開いて、エラーメッセージがないか確認してください
4. ブラウザの自動再生ポリシーにより、最初のクリックまで再生されない場合があります

### 対応ブラウザ

- Chrome / Edge: MP3, OGG
- Firefox: MP3, OGG
- Safari: MP3
